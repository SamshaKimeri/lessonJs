Les Variables

En informatique, une variable est un conteneur qui stocke la donnée temporairement au sein de votre code.

Cela vous permet d’enregistrer des données.

Il existe des types basiques :

- number : la donnée est un chiffre ;

- string : la donnée est une chaîne de caractères

(du texte, comme “David”, par exemple) ;

- boolean : la donnée est vraie ou fausse

Utiliser des opérateurs

Un opérateur est un symbole qui permet d’effectuer des actions sur des variables ou des valeurs

Ces actions peuvent être des opérations mathématiques, ou encore des comparaisons entre valeurs.

Les opérateurs d'affectation

Lorsqu’on utilise += , on additionne la valeur à droite du signe = à notre variable.

Les variantes pour les autres opérations mathématiques existent également.

On aura donc -= pour une soustraction, *= pour une multiplication, /= pour une division.

L'instruction Number

nous pouvons soit enlever les guillemets autour des chiffres,

soit préciser à JavaScript que nous voulons vraiment utiliser des chiffres,

grâce à l’instruction Number

Cette dernière permet de convertir le texte “150” en chiffre 150.

Les Object en JS

Un Object (objet, en français) JavaScript est un conteneur.

Il est composé de propriétés qui ont chacune une valeur

Ainsi, le type de donnée Object offre la possibilité de stocker plusieurs valeurs en une seule fois,

plutôt que de devoir stocker séparément nos valeurs dans plusieurs variables différentes.

Exemple :

Pour déclarer un objet en JavaScript, vous devez utiliser les accolades { } .

Les propriétés d’un objet ont un nom et une valeur qui sont assignées avec deux points:

Les propriétés sont séparées entre elles par des virgules,.

Par exemple :

let monPerso = {

nom : "Wayne",

prenom : "Bruce",

}

dans l'exemple nom et prenom sont des propriétes

et Wayne et Bruce sont des valeurs

Ajouter une propriété à un object

Un objet JavaScript peut être mis à jour au fur et à mesure de son évolution dans le code.

Par exemple, si vous vous rendez compte que l’objet monPerso a désormais un véhicule préféré

il vous faudra lui ajouter cette nouvelle propriété.

Pour ajouter une propriété, vous devez écrire le nom de votre objet,

séparé par un point . avec le nom de votre nouvelle propriété :

monPerso.vehiculePrefere = "Batmobile"

Accédez à une propriété d’un objet JavaScript

Lorsque vous manipulez des objets JavaScript,

vous avez également besoin d’accéder à leurs propriétés pour les stocker dans des variables,

et les utiliser dans la suite de votre code ou dans un autre contexte.

Vous pouvez accéder à la valeur d’une propriété en utilisant le point .

const nomDeMonPersonnage = monPerso.nom

// Vérification

console.log(nomDeMonPersonnage)

console.log(monPerso.nom)

Regrouper des données grâce aux tableaux

Un tableau en JavaScript est donc un objet qui permet de lister

plusieurs variables ou valeurs, et de les regrouper.

Pour accéder à un élément de votre tableau,

vous devez utiliser les crochets,

et mettre entre crochets le numéro de la case que vous voulez regarder.

La première case du tableau correspond à la case numéro zéro.

Comptez le nombre d’éléments de votre tableau

Pour connaître le nombre d’éléments que contient votre tableau,

vous devez accéder à la propriétélength, en utilisant le point . , comme pour les objets.

Modifier des tableaux grace aux méthodes

Les tableaux sont des conteneurs, comme les objets que nous avons vus dans le chapitre précédent.

Pour effectuer des actions sur les données stockées dans votre tableau

(ajouter, supprimer des données…), vous utiliserez des méthodes

Les méthodes s’utilisent avec des parenthèses ( )

À l’intérieur de ces parenthèses vous pouvez passer des valeurs,

c'est-à-dire des données, qui serviront à la méthode pour modifier les données de votre tableau.

La méthode .push()

Pour ajouter des données à un tableau,

vous devez utiliser la méthode push en lui indiquant la valeur que vous souhaitez ajouter.

let mesFilms = ["Titanic", "Jurassic Park"]

mesFilms.push("Retour vers le futur")

console.log(mesFilms)

// mesFilms vaut ["Titanic", "Jurassic Park", "Retour vers le Futur"]

La méthode .pop()

Pour supprimer la dernière donnée de votre tableau,

vous pouvez utiliser la méthode pop sans avoir besoin de passer de valeurs .

(c'est-à-dire, sans rien écrire entre les parenthèses)

Exemple :

let mesFilms = ["Titanic", "Jurassic Park", "Retour vers le futur"]

mesFilms.pop()

// mesFilms vaut ["Titanic", "Jurassic Park"]

La copie d'une variable par VALEUR

il arrive parfois que l’on veuille dupliquer une variable.

C’est le cas lorsque l’on doit sauvegarder une valeur avant de la modifier, par exemple.

le plus simple est de copier le contenu d’une variable à l’intérieur d’une autre variable.

Nous avons copié le contenu d’une variable à l’intérieur d’une autre variable.

Nous avons donc deux variables indépendantes.

La copie d'une variable par REFERENCE

copier une variable qui contient un contenu de type “complexe” :

un objet ou un tableau, par exemple.

Dans ce cas, JavaScript fait une copie par référence

Exemple :

let variableComplexe1 = ['pomme', 'cerise']

let variableComplexe2 = variableComplexe1

variable1 et variable2 sont deux étiquettes différentes

c’est l’étiquette qui a été copiée.

La copie d'un objet ou d'un tableau

Utiliser le spread operator (...) pour les tableaux ou les objets

Le spread operator (...) crée une copie superficielle de l'objet ou du tableau

ce qui signifie qu'il copie les valeurs des propriétés ou des éléments,

mais s'il existe des objets imbriqués, ceux-ci ne seront pas copiés par valeur.

// Pour un tableau

let tableauOriginal = [1, 2, 3];

let copieTableau = [...tableauOriginal]; // Copie par valeur

copieTableau.push(4);

console.log(tableauOriginal); // Affiche : [1, 2, 3]

console.log(copieTableau); // Affiche : [1, 2, 3, 4]

// Pour un objet

let objetOriginal = { a: 1, b: 2 };

let copieObjet = { ...objetOriginal }; // Copie par valeur

copieObjet.c = 3;

console.log(objetOriginal); // Affiche : { a: 1, b: 2 }

console.log(copieObjet); // Affiche : { a: 1, b: 2, c: 3 }

Les expressions régulières

\d : Cela représente un chiffre. En termes simples, il correspond à n'importe quel

caractère numérique de 0 à 9.

+ : Cela indique que le chiffre (représenté par \d) doit apparaître une ou plusieurs fois.

Cela signifie que la séquence de chiffres doit être continue.

/g : C'est un modificateur global qui signifie "global". Il indique à l'expression régulière

de rechercher toutes les occurrences de la correspondance dans la chaîne, pas seulement

la première.

Exemple :

let chaine = "abc123def456";
let nouvelleChaine = chaine.replace(/\d+/g, "X");
console.log(nouvelleChaine);
Résultat : "abcXdefX"

Dans cet exemple, les séquences de chiffres "123" et "456"

sont remplacées par la lettre "X",

produisant la chaîne résultante "abcXdefX".

Autres expressions régulières

[a-zA-Z] : Cela correspond à n'importe quelle lettre majuscule ou minuscule de l'alphabet

anglais. L'utilisation de a-z indique les lettres minuscules et A-Z indique les lettres

majuscules.

[^a-zA-Z] : En ajoutant ^ au début de l'ensemble, cela inverse la correspondance.

Donc, cela signifie tout caractère qui n'est pas une lettre majuscule ou minuscule

de l'alphabet anglais.

Exemple :

let chaine = "Hello 123 World!";
let nouvelleChaine = chaine.replace(new RegExp("[^a-zA-Z]", 'g'), "");
console.log(nouvelleChaine);
// Résultat : "HelloWorld"

Autres expressions régulières

/ /g est utilisée pour rechercher tous les espaces dans une chaîne de caractères.

/ / : Cela représente un espace simple. Dans une expression régulière,

les caractères spéciaux comme l'espace peuvent être inclus

directement s'ils ne nécessitent pas d'échappement.

/ /g : L'ajout du modificateur global g à la fin de l'expression régulière

indique une recherche globale.

Cela signifie que l'expression régulière va rechercher toutes les occurrences

de l'espace dans la chaîne, pas seulement la première occurrence.

let chaine = "Bonjour le monde !";
let nouvelleChaine = chaine.replace(/ /g, "_");
console.log(nouvelleChaine);
// Résultat : "Bonjour_le_monde_!"

Utiliser REPLACE pour remplacer toutes les occurrences de lettre1 par lettre2
let nouvelleChaine = chaine.replace(new RegExp(lettre1, "g"), lettre2);

La méthode replace() ne modifie pas la chaîne originale,

mais renvoie une nouvelle chaîne modifiée.

regexp : Une expression régulière ou une sous-chaîne à rechercher dans la chaîne d'origine.
substr : La sous-chaîne à rechercher dans la chaîne d'origine.
newSubstr : La sous-chaîne qui remplacera la correspondance.
